01 Vous en utilisez sans le savoir

Le harnais le plus courant ? Celui que vous avez ouvert ce matin.

Avant de partir chercher des outils exotiques, regardons ce que vous avez probablement utilisé aujourd'hui. Quand un chatbot va sur le web, lit un fichier, ou exécute du code que vous lui avez demandé — il n'y a pas que le modèle au travail. Il y a un harnais autour.

Le moment « ah mais alors… »

Quand vous demandez à ChatGPT de chercher quelque chose sur le web, ou à Claude de lire un PDF que vous venez de glisser dans la conversation, vous activez un harnais.

Le modèle, seul, ne peut ni naviguer ni lire de fichier. C'est l'application qui l'entoure — la boucle, les outils, la mémoire de conversation — qui transforme ces capacités en réalité.

Vous avez déjà l'expérience

Si vous avez déjà vu un assistant IA réfléchir, agir, observer, recommencer — vous avez vu un harnais à l'œuvre. Le concept n'est pas réservé aux ingénieurs : c'est partout, dans tous les produits qui vous laissent déléguer une tâche plutôt que de simplement poser une question.

Vous avez probablement croisé :
ChatGPT (mode browse) Claude.ai (recherche, fichiers) Perplexity Gemini (avec extensions)
02 Quatre familles de harnais

Le panorama des harnais en 2026

Les produits agentiques se déclinent en quatre grandes familles, selon ce qu'ils permettent au modèle de toucher : le code, votre boîte mail, votre calendrier, le web, ou simplement votre conversation. Chacun suit la même recette : un modèle au cœur, un harnais autour.

Famille 1 · Intégrés au chat

Les harnais que vous utilisez déjà

Les chatbots grand public ont tous un harnais intégré — c'est lui qui leur permet d'aller sur le web, de lire vos fichiers, ou d'exécuter du code à votre demande.
ChatGPT
OpenAI

Avec ses modes Browse, Code Interpreter, et les GPTs, ChatGPT n'est plus un simple chatbot : c'est un agent qui peut chercher sur le web, exécuter du Python, lire vos fichiers et appeler des APIs.

grand public web · code · fichiers
Claude.ai
Anthropic

L'interface web de Claude inclut un harnais qui orchestre la recherche web, la lecture de documents, l'exécution de code et la création d'artefacts interactifs — comme cette page que vous lisez en ce moment.

grand public recherche · artefacts
Perplexity
Perplexity AI

Spécialisé dans la recherche d'information : son harnais coordonne plusieurs requêtes web, lit les pages trouvées, et synthétise une réponse sourcée. C'est un harnais simple mais très ciblé.

recherche sources · synthèse
Famille 2 · IDE & terminal

Harnais pour développeurs

L'agent vit dans votre éditeur ou votre terminal. Il lit votre code, le modifie, exécute des commandes — vous restez aux commandes, mais il fait le gros du travail.
Claude Code
Anthropic

Un harnais en ligne de commande. Claude lit toute votre base de code, modifie les fichiers, lance les tests, gère votre git. Anthropic le décrit explicitement comme « the agentic harness around Claude ».

terminal git · MCP
Cursor
Anysphere

Un fork de VS Code transformé en IDE agentique. Le mode Composer permet à l'agent d'éditer plusieurs fichiers à la fois ; les Background Agents travaillent en arrière-plan pendant que vous codez.

IDE multi-modèles
Antigravity
Google

L'IDE agent-first de Google. Il pilote plusieurs agents en parallèle qui ont accès à l'éditeur, au terminal et au navigateur Chrome. Un excellent exemple de harnais multi-agent.

IDE multi-agent · navigateur
OpenCode
Open source

Un harnais open-source, agnostique en modèle : il fonctionne avec Claude, GPT, Gemini ou un modèle local. Pour ceux qui veulent comprendre, modifier, ou héberger leur propre agent de codage.

open source tout modèle
Codex CLI
OpenAI

La réponse d'OpenAI à Claude Code, en ligne de commande, et open-source. Construit sur GPT et conçu pour les workflows asynchrones. Code source disponible pour étudier la mécanique d'un harnais.

terminal open source
GitHub Copilot Workspace
Microsoft

L'évolution agentique de Copilot. Vous pointez une issue GitHub, il planifie, modifie le code à travers le dépôt, et propose une PR. Intégré au workflow GitHub que des millions de devs utilisent déjà.

github issues · PR
Famille 3 · Asynchrone

Harnais qui travaillent pendant que vous dormez

Vous donnez une mission, vous fermez votre laptop. L'agent travaille seul dans le cloud et vous rend un résultat — typiquement une pull request — quelques minutes ou quelques heures plus tard.
Devin
Cognition AI

L'archétype de l'« ingénieur logiciel autonome ». Vous lui donnez un ticket Jira ou Linear, il configure son environnement, code, teste, et ouvre une pull request. Plutôt orienté grandes équipes.

cloud jira · linear · PR
Jules
Google

L'équivalent de Devin chez Google, plus accessible. Connecté à GitHub, il clone votre dépôt dans une VM Google Cloud, travaille seul, et propose une PR. Construit comme un système multi-agent en interne.

cloud github · multi-agent
Replit Agent
Replit

Un harnais qui ne se contente pas d'écrire du code : il déploie l'application complète. Vous décrivez une idée d'app, l'agent la construit, la teste, et la met en ligne.

cloud app · déploiement
Famille 4 · Personnel

Harnais pour le quotidien (non-code)

L'agent ne touche pas à votre code mais à vos messages, votre calendrier, vos fichiers, votre maison. Le rêve du « majordome IA » qui prend forme depuis 2025.
Clawdbot / OpenClaw
Peter Steinberger

Un harnais auto-hébergé qui connecte Claude (ou un autre modèle) à WhatsApp, Telegram, Discord, votre Mac, votre email, votre domotique. L'auteur le décrit comme « Claude with hands ».

auto-hébergé messageries · domotique
Claude pour Chrome
Anthropic

Un harnais navigateur : Claude voit ce que vous voyez, peut cliquer, remplir des formulaires, naviguer entre les onglets. Il transforme n'importe quelle page web en interface manipulable par un agent.

navigateur clics · formulaires
Cowork
Anthropic

Un harnais pour automatiser les tâches bureautiques : tri d'emails, mise en forme de documents, manipulation de fichiers. Pensé pour les utilisateurs non-techniques qui veulent déléguer du travail répétitif.

bureautique emails · fichiers
03 À retenir

Le harnais est partout, dès qu'il y a action

Vous l'aurez remarqué : derrière des produits qui semblent très différents — un IDE, un chatbot, un majordome WhatsApp — la recette est la même. Un modèle au cœur, des outils branchés autour, une boucle qui rappelle le modèle après chaque action, une mémoire qui maintient le fil. C'est ça, un harnais.

La grande compétition de l'industrie en 2026 n'est plus seulement sur les modèles. Tous les acteurs majeurs ont accès à des modèles excellents. Ce qui les différencie, c'est la qualité du harnais qu'ils construisent autour : son ergonomie, ses outils, sa fiabilité, sa capacité à enchaîner les actions sans perdre le fil.

La règle qui explique tout

Le modèle, c'est l'intelligence. Le harnais, c'est ce qui lui donne des mains.

La qualité d'un produit agentique se joue beaucoup plus dans le harnais que dans le modèle qu'il utilise.
